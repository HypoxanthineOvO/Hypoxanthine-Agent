# 心跳检查清单

你每次被唤醒时，请按以下清单自主检查并汇报。

## 服务器状态
- 用 run_command 检查 CPU、内存、GPU 使用率
- 如果有长时间高消耗进程（>30min, >50% CPU），汇报

## 邮件
- 调用 scan_emails 扫描未读邮件
- 🔴 重要邮件：展开摘要，附件信息放在概要下方
- ⚪ 普通邮件：一句话概括

## 提醒
- 调用 list_reminders 检查有没有漏网的过期提醒

## 汇报规则
- 有事：统一汇报一条消息，语气符合你的人设
- 无事：严格静默，输出且只输出下面这一行（必须完全一致，不要加任何其他文字、空格、标点、emoji，不要加引号/代码块）：
**SILENT**
