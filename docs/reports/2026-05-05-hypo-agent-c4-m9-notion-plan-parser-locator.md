# C4 M9 - 日程解析与插入定位器

## 结果

- 新增 `src/hypo_agent/core/notion_plan_editor.py`。
- 支持单条、多条、默认年份、无时间、跨天写开始日期。
- 同日按开始时间升序，无时间事项排到有时间事项之后。
- 支持缺日期块 preview，缺月份页会生成创建计划。

## 已覆盖案例

- `5/8 10:30-11:30 普拉提训练` -> `2026-05-08`。
- `5/8 整理材料` -> 无时间，排在当天有时间事项之后。
- `5/8 22:00-5/9 01:00 写作` -> 写在 `5/8`，保留完整时间范围。
- 当新事项早于当天第一条时，插入 anchor 是日期 heading，避免追加到月份页末尾。
